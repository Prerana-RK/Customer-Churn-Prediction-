# Customer Churn Prediction 📊

This project aims to predict customer churn for a telecommunications company using Machine Learning. By analyzing historical customer data, the model identifies patterns that lead to attrition, allowing the business to take proactive retention measures.

## 🚀 Project Overview
The project is implemented in a Google Colab notebook (coustomer churn prediction.ipynb) and follows a standard data science pipeline:
- *Data Loading*: Importing the Telco Customer Churn dataset.
- *Data Exploration (EDA)*: Analyzing features like gender, SeniorCitizen, tenure, and MultipleLines to understand their impact on churn.
- *Preprocessing*: Handling missing values and preparing categorical data for machine learning.
- *Modeling*: Training a predictive model to classify whether a customer will leave (Churn) or stay.

## 📂 Dataset Details
The dataset contains information on 7,043 customers and includes 21 features:
- *Demographics*: Gender, Senior Citizen status, Partner, and Dependents.
- *Services*: Phone service, Multiple lines, Internet service (DSL/Fiber optic), Online security, etc.
- *Account Info*: Tenure, Contract type, Payment method, and Charges.
- *Target Variable*: Churn (Yes/No).
